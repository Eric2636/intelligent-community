# 公众平台配置清单（提交审核前必做）

以下均在 [微信公众平台](https://mp.weixin.qq.com) 登录你的小程序后操作。

---

## 一、小程序类目

1. 进入 **设置** → **基本设置** → **服务类目**。
2. 结合「智能社区」功能，建议选择以下类目（至少选前两项）：

| 功能 | 建议类目 | 说明与资质 |
|------|----------|------------|
| **社区论坛**（发帖、回复、UGC） | **社交** → **社区/论坛** | 适用于论坛、贴吧、社群等 UGC 发布与交流。需（2选1）：《非经营性互联网信息服务备案核准》或机关《统一社会信用代码证》。**首次提交代码审核需经当地互联网主管机关审核，约 7 天。** |
| **任务互助**（发布/领取任务、跑腿类） | **生活服务** → **跑腿** | 适用于委托跑腿、代为购买和送达等。**无额外资质要求。** |
| **任务互助**（预约服务类） | **生活服务** → **家政服务** | 适用于保姆、月嫂、保洁、家电维修等在线预约。**无额外资质要求。** 与「跑腿」二选一即可，选更贴近你产品描述的。 |
| **二手商城**（社区闲置交易） | 见下方说明 | 见「二手类目说明」。 |

3. **二手类目说明**  
   - 若以**信息展示 + 线下/站内成交**为主（无多商家入驻、无典型电商平台形态）：可先只选 **社交-社区/论坛** + **生活服务-跑腿（或家政）**，在版本描述中写「含社区内闲置物品信息展示与联系，交易由用户线下/自行完成」。审核若要求补充再补。  
   - 若明确是**多卖家入驻的 C2C 交易平台**：需选 **电商平台** → **电商平台**，并具备《增值电信业务经营许可证》等平台类资质。

4. 确保所选类目与「任务互助、社区论坛、二手商城」等描述一致，避免类目不符被拒。

---

## 二、用户隐私保护指引

1. 进入 **设置** → **基本设置**。
2. 找到 **用户隐私保护指引** / **服务内容声明**（入口名称以当前后台为准）。
3. 按页面要求填写：
   - 若你有**官网或 H5 的隐私政策页**：填写该页面的 URL。
   - 若没有外链：按平台提示填写说明（如「隐私政策见小程序内《隐私政策》页面」等，以当前页面选项为准）。
4. 保存后等待生效（如有审核则按提示完成）。

---

## 三、订阅消息（可选，已配置模板 ID 则跳过）

1. 进入 **功能** → **订阅消息** → **公共模板库**。
2. 选用模板（如：订单发起提醒、订单结束服务通知等），获取 **模板 ID**。
3. 将模板 ID 填入项目 **`config/index.js`** 的 `subscribeTemplateIds` 数组。
4. 当前项目已配置 2 个模板 ID，若你新增或更换了模板，记得同步修改 `config/index.js`。

---

## 四、自检打勾

- [ ] 服务类目已选且与小程序功能一致
- [ ] 用户隐私保护指引已填写并保存
- [ ] 订阅消息模板 ID（若有变更）已同步到 `config/index.js`

完成以上后，再配合《发布前自测清单》做小程序内验证，即可提交审核。
