# 隐私与协议验证说明

## 一、小程序内验证（必做）

1. **登录页**
   - 打开登录页，确认有「同意《用户协议》和《隐私政策》」勾选项。
   - 点击 **《用户协议》**，应跳转到「用户协议」页并展示完整协议内容。
   - 点击 **《隐私政策》**，应跳转到「隐私政策」页并展示完整隐私政策内容。
   - 未勾选时点击「微信授权登录」，应提示「请先同意用户协议」且不能登录。
   - 勾选后再登录，应能正常完成登录。

2. **协议内容**
   - 在 `packageCommon/agreement/index.js` 的 `content` 中维护**用户协议**正文。
   - 在 `packageCommon/privacy/index.js` 的 `content` 中维护**隐私政策**正文。
   - 可根据实际业务修改文案（收集哪些信息、用途、存储方式、用户权利等）。

---

## 二、微信公众平台侧（提交审核前）

1. **用户隐私保护指引**
   - 登录 [微信公众平台](https://mp.weixin.qq.com) → **设置** → **基本设置**。
   - 找到「服务内容声明」或「用户隐私保护指引」相关入口。
   - 填写**隐私政策链接**：若你有官网或 H5 的隐私政策页，填该 URL；没有则可先填小程序内展示的说明（部分类目支持填写「见小程序内隐私政策」等说明，以平台当前规则为准）。

2. **小程序类目**
   - 确保所选类目与「任务 / 论坛 / 二手商城」等实际功能一致，避免因类目不符被拒。

3. **审核说明**
   - 提交审核时可在「版本描述」中注明：已在小程序内提供用户协议与隐私政策，登录前需勾选同意。

---

## 三、自测检查清单

- [ ] 登录页《用户协议》可点击并能打开协议页
- [ ] 登录页《隐私政策》可点击并能打开隐私政策页
- [ ] 未勾选协议无法登录，勾选后可正常登录
- [ ] 协议与隐私政策正文无错别字、无与当前功能明显不符的描述
- [ ] 公众平台已按要求填写隐私保护指引（若当前类目要求）

完成以上即完成「隐私与协议」的验证与配置。
