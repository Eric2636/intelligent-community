# 数据库集合创建指南

## 需要创建的7个数据库集合

请在微信开发者工具的云开发控制台中手动创建以下集合：

### 创建步骤：
1. 打开微信开发者工具
2. 点击顶部"云开发"按钮
3. 在左侧菜单中找到"数据库"
4. 点击"添加集合"按钮
5. 输入集合名称，点击确定

### 需要创建的集合：

1. **tasks** - 任务集合
2. **posts** - 帖子集合
3. **replies** - 回复集合
4. **items** - 商品集合
5. **users** - 用户集合
6. **notifications** - 通知集合
7. **feedbacks** - 反馈集合

### 权限设置：
- 默认权限：仅创建者可读写

### 验证创建成功：
创建完成后，在数据库页面应该能看到这7个集合。

## 如果还是看不到数据

集合创建成功后，数据会在你实际使用小程序功能时自动生成。
- 访问任务页面 → tasks集合会有数据
- 发布帖子 → posts集合会有数据
- 等等...
