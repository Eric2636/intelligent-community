# 智能社区小程序 - 快速启动指南

## 🚀 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 微信开发者工具配置
1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择项目根目录：`/Users/chenglingyun/Documents/fuye-project/intelligent community`
4. AppID 选择"测试号"或使用自己的 AppID
5. 点击"导入"

### 3. 构建 npm 包
1. 在微信开发者工具中，点击菜单"工具" → "构建 npm"
2. 等待构建完成

### 4. 预览项目
点击微信开发者工具右上角的"预览"按钮，使用手机扫码预览

## 📱 功能导航

### 主要功能入口（底部 TabBar）
1. **任务** - 任务中心，查看和发布社区任务
2. **论坛** - 社区论坛，发帖和交流
3. **商城** - 二手商城，买卖闲置物品
4. **我的** - 个人中心，管理个人信息和功能

### 详细功能说明

#### 📋 任务中心
- **查看任务**: 浏览待领取的任务列表
- **发布任务**: 点击右下角"+"按钮发布新任务
- **领取任务**: 点击任务卡片领取任务
- **我的任务**: 在"我的"页面查看"我的任务"

#### 💬 社区论坛
- **浏览帖子**: 查看置顶帖和最新帖子
- **发帖**: 点击右下角"+"按钮发布新帖
- **回复**: 点击帖子查看详情，发表回复
- **我的帖子**: 在"我的"页面查看"我的帖子"

#### 🛒 二手商城
- **浏览商品**: 按分类浏览二手商品
- **发布商品**: 点击右下角"+"按钮发布商品
- **查看详情**: 点击商品查看详细信息
- **我的商品**: 在"我的"页面查看"我发布的"

#### 👤 个人中心
- **登录/注册**: 点击头像进行登录
- **编辑信息**: 点击头像进入个人信息编辑
- **功能菜单**: 各种功能统一入口
  - 我的任务
  - 我的帖子
  - 我发布的（商城）
  - 消息通知
  - 意见反馈
  - 关于我们
  - 设置

## 🎯 快速体验流程

### 1. 体验任务功能
1. 点击底部"任务"Tab
2. 查看任务列表
3. 点击右下角"+"发布一个任务
4. 填写任务信息（标题、说明、佣金）
5. 点击发布
6. 在列表中查看发布的任务
7. 点击任务进入详情
8. 点击"领取任务"
9. 在任务详情中提交完成证明
10. 在"我的" → "我的任务"中查看

### 2. 体验论坛功能
1. 点击底部"论坛"Tab
2. 查看置顶帖和最新帖
3. 点击右下角"+"发布一个帖子
4. 填写标题和内容
5. 点击发布
6. 在列表中查看发布的帖子
7. 点击帖子进入详情
8. 在底部输入框发表回复
9. 在"我的" → "我的帖子"中查看

### 3. 体验商城功能
1. 点击底部"商城"Tab
2. 切换分类标签浏览商品
3. 点击右下角"+"发布商品
4. 选择分类，填写商品信息
5. 点击发布
6. 在列表中查看发布的商品
7. 点击商品查看详情
8. 点击"联系TA"
9. 在"我的" → "我发布的"中查看

### 4. 体验个人中心
1. 点击底部"我的"Tab
2. 点击头像进入登录
3. 选择手机号验证码登录或密码登录
4. 输入信息完成登录
5. 点击头像编辑个人信息
6. 修改昵称、性别、生日、地址等
7. 点击保存
8. 点击各个功能菜单体验

## 📊 项目结构

```
intelligent community/
├── pages/              # 主包页面（Tab页）
│   ├── task/          # 任务中心
│   ├── forum/         # 社区论坛
│   ├── mall/          # 二手商城
│   └── my/            # 个人中心
├── packageTask/       # 任务分包
│   ├── detail/        # 任务详情
│   ├── publish/       # 发布任务
│   └── my-tasks/      # 我的任务
├── packageForum/      # 论坛分包
│   ├── post/          # 帖子详情
│   ├── publish/       # 发布帖子
│   └── my-posts/      # 我的帖子
├── packageMall/       # 商城分包
│   ├── detail/        # 商品详情
│   ├── publish/       # 发布商品
│   └── my-list/       # 我的商品
├── packageCommon/     # 公共功能分包
│   ├── notice/        # 消息通知
│   ├── feedback/      # 意见反馈
│   └── about/         # 关于我们
├── custom-tab-bar/    # 自定义TabBar
├── mock/              # Mock数据
├── components/        # 自定义组件
├── static/           # 静态资源
└── utils/            # 工具函数
```

## 🎨 UI 组件使用

项目使用 TDesign 小程序组件库，常用组件：
- `t-button` - 按钮
- `t-input` - 输入框
- `t-textarea` - 文本域
- `t-cell` - 单元格
- `t-card` - 卡片
- `t-icon` - 图标
- `t-tag` - 标签
- `t-loading` - 加载
- `t-toast` - 提示
- `t-empty` - 空状态

更多组件参考：[TDesign 小程序组件库文档](https://tdesign.tencent.com/miniprogram/components/overview)

## ⚠️ 注意事项

1. **禁止修改的文件**
   - `project.config.json` - 项目配置文件
   - AppID - 应用ID（如需修改请联系管理员）

2. **数据存储**
   - 当前使用 Mock 数据，刷新后数据会重置
   - 登录 token 存储在本地，清除缓存后需要重新登录

3. **兼容性**
   - 最低基础库版本：2.6.5
   - 建议使用最新版本的微信开发者工具
